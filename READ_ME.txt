You need the DLL in the opencv_world341.zip file
1- Run "Probabl_Hough.exe"
2- Press "FILE" menu and select a an image
3- Choose FILE->Canny
4- Either proceed to Morphological Operators or select TRANSFORM->(prob or Circular Hough)

Observe the results.

You can change the parameters of the HoughLines funtion from the GUI. (Not Hough Circles)
